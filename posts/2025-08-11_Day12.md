<p>1. 메서드란&nbsp;&nbsp; <br />-&nbsp;class&nbsp;내부에&nbsp;정의된&nbsp;함수(Function)를&nbsp;메서드(Method)라고&nbsp;부름&nbsp;&nbsp; <br />-&nbsp;특정&nbsp;기능을&nbsp;수행하는&nbsp;코드블록을&nbsp;정의하고,&nbsp;필요할&nbsp;때&nbsp;호출하여&nbsp;재사용&nbsp;가능&nbsp;&nbsp; <br />-&nbsp;중복&nbsp;코드&nbsp;제거와&nbsp;가독성,&nbsp;유지&nbsp;보수성이&nbsp;향상됨&nbsp;&nbsp;</p>
<p>&nbsp;</p>
<p>[기본구조]&nbsp;</p>
<pre class="bash" id="code_1754894660210"><code>[접근제한자] [static] 반환형 메서드 이름(매개변수){  
    // 함수에서 실행할 코드  
    [return 값]; // 반환형이 void면 return을 생략하거나 return;로 메서드 종료 가능  
}</code></pre>

<p>2. 지역변수</p>
<p>- 블록({}) 내부에서 선언된 변수</p>
<p>- 해당 블록을 벗어나면 메모리에서 제거됨</p>
<p>- 선언 시 반드시 초기화 후 사용해야 함</p>
<p>&nbsp;</p>
<p>3. 매개변수<br />- 메서드 호출 시 외부에서 전달받는 값을 저장하는 지역변수<br />- 메서드 선언부에 정의됨<br />- 메서드 실행 시 생성되고, 메서드 종료 시 사라짐</p>
<p>&nbsp;</p>
<p>4. Call&nbsp;By&nbsp;Value(값에&nbsp;의한&nbsp;호출)&nbsp;&rarr;&nbsp;자바는&nbsp;무조건&nbsp;이&nbsp;방식으로&nbsp;호출 <br />-&nbsp;메모리에&nbsp;담긴&nbsp;값&nbsp;자체를&nbsp;전달 <br />-&nbsp;기본형&nbsp;&rarr;&nbsp;값(데이터)&nbsp;자체가&nbsp;복사됨 <br />-&nbsp;참조형&nbsp;&rarr;&nbsp;참조값(주소)이&nbsp;복사됨 <br />-&nbsp;주소값을&nbsp;복사해서&nbsp;같은&nbsp;객체를&nbsp;가리키게&nbsp;되므로,&nbsp;내부&nbsp;데이터를&nbsp;변경하면&nbsp;원본에도&nbsp;영향을&nbsp;줌</p>
<p>&nbsp;</p>
<p>5. 메서드&nbsp;오버로딩 <br />-&nbsp;오버로딩:&nbsp;같은&nbsp;이름의&nbsp;메서드를&nbsp;매개변수의&nbsp;개수,&nbsp;타입,&nbsp;순서를&nbsp;다르게&nbsp;하여&nbsp;여러&nbsp;개&nbsp;정의하는&nbsp;것 <br />-&nbsp;반환형(return&nbsp;type)은&nbsp;오버로딩의&nbsp;구분&nbsp;기준이&nbsp;아님 <br />-&nbsp;함수의&nbsp;이름&nbsp;=&nbsp;기능의&nbsp;이름 <br />-&nbsp;매개변수&nbsp;=&nbsp;함수에서&nbsp;이용하는&nbsp;데이터 <br />-&nbsp;반환형&nbsp;=&nbsp;결과</p>
<p><br />6. 오버로딩을 사용하는 목적 <br />-&nbsp;동일한&nbsp;기능을&nbsp;다양한&nbsp;입력&nbsp;형태로&nbsp;편리하게&nbsp;사용하기&nbsp;위해서</p>

<p>배운점 &amp; 느낀점</p>
<p>이번에&nbsp;2차원&nbsp;배열과&nbsp;메서드&nbsp;관련&nbsp;내용을&nbsp;학습하면서,&nbsp;배열의&nbsp;구조와&nbsp;메서드의&nbsp;활용법을&nbsp;좀&nbsp;더&nbsp;명확하게&nbsp;이해하게&nbsp;되었다.&nbsp;특히&nbsp;2차원&nbsp;배열은&nbsp;단순히&nbsp;&ldquo;표&rdquo;처럼&nbsp;생각하면&nbsp;쉽게&nbsp;접근할&nbsp;수&nbsp;있었고,&nbsp;행과&nbsp;열의&nbsp;인덱스를&nbsp;통해&nbsp;모든&nbsp;요소를&nbsp;탐색할&nbsp;수&nbsp;있다는&nbsp;점이&nbsp;인상&nbsp;깊었다.&nbsp;또한&nbsp;서로&nbsp;다른&nbsp;길이의&nbsp;1차원&nbsp;배열을&nbsp;묶을&nbsp;수&nbsp;있다는&nbsp;점도&nbsp;유연하게&nbsp;활용할&nbsp;수&nbsp;있겠다고&nbsp;느꼈다. <br />메서드&nbsp;부분에서는&nbsp;코드의&nbsp;재사용성과&nbsp;가독성을&nbsp;높이는&nbsp;데&nbsp;중요한&nbsp;역할을&nbsp;한다는&nbsp;것을&nbsp;다시&nbsp;한&nbsp;번&nbsp;깨달았다.&nbsp;지역변수와&nbsp;매개변수의&nbsp;차이,&nbsp;그리고&nbsp;자바의&nbsp;호출&nbsp;방식이&nbsp;무조건&nbsp;Call&nbsp;By&nbsp;Value라는&nbsp;점은&nbsp;실제&nbsp;코드&nbsp;작성&nbsp;시&nbsp;실수를&nbsp;줄이는&nbsp;데&nbsp;도움이&nbsp;될&nbsp;것&nbsp;같다. <br />오버로딩&nbsp;개념을&nbsp;통해&nbsp;같은&nbsp;기능을&nbsp;다양한&nbsp;형태로&nbsp;구현할&nbsp;수&nbsp;있다는&nbsp;점이&nbsp;편리하다고&nbsp;느꼈고,&nbsp;단순히&nbsp;매개변수&nbsp;이름이&nbsp;다른&nbsp;것만으로는&nbsp;오버로딩이&nbsp;성립하지&nbsp;않는다는&nbsp;점도&nbsp;기억에&nbsp;남았다.&nbsp;전반적으로&nbsp;이번&nbsp;학습을&nbsp;통해&nbsp;배열과&nbsp;메서드에&nbsp;대한&nbsp;이해도가&nbsp;한층&nbsp;높아졌고,&nbsp;이를&nbsp;바탕으로&nbsp;더&nbsp;깔끔하고&nbsp;유지보수하기&nbsp;좋은&nbsp;코드를&nbsp;작성할&nbsp;수&nbsp;있을&nbsp;것&nbsp;같다.</p>
