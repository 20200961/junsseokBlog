<h3 style="text-align: left;">전자 정부 프레임워크(eGovFrame)란?</h3>
<p>- 한국 정부가 공공기관 시스템 개발을 쉽게 할 수 있도록 만들어 둔 표준 개발 프레임워크입니다.</p>
<p>- 웹 시스템을 만들 때 개발 방식과 구조를 표준화한 개발 틀입니다.</p>
<p>- 개발자가 처음부터 모든 구조를 만드는 것이 아니라 이미 만들어진 기본 구조 + 라이브러리 + 규칙을 사용하여 개발합니다.</p>
<hr contenteditable="false" />
<h3>생겨난 이유</h3>
<p>이전에는 공공기관 시스템을 만들 때 회사마다 개발 방식이 다 다르기 때문에 유지보수가 어려웠습니다.</p>
<p>따라서 공공 시스템 개발 표준이 생기게 되어습니다.</p>
<p>&nbsp;</p>
<p>- 공공 시스템 표준화</p>
<p>- 개발 생산성 향상</p>
<p>- 유지보수 용이하게&nbsp;</p>
<p>- 특정 업체 종속 방지</p>
<p>이와 같은 장점이 있습니다.</p>
<hr contenteditable="false" />

<h3>예시 코드</h3>
<p>회원 조회 기능</p>
<p>&nbsp;</p>
<p><b>1. Controller(요청 받는 곳)</b></p>
<p>사용자가 URL로 요청</p>
<pre class="bash" id="code_1773470959187"><code>@Controller
public class MemberController {

    @Resource(name="memberService")
    private MemberService memberService;

    @RequestMapping("/member/list.do")
    public String memberList(Model model) throws Exception {

        List&lt;MemberVO&gt; memberList = memberService.selectMemberList();

        model.addAttribute("memberList", memberList);

        return "member/memberList";
    }
}</code></pre>
<p>- 웹 요청 받음</p>
<p>- Service 호출</p>
<p>- 화면(JSP)에 데이터 전달</p>
<p>&nbsp;</p>
<p><b>2. Service(비즈니스 로직)</b></p>
<pre class="bash" id="code_1773471014676"><code>@Service("memberService")
public class MemberServiceImpl implements MemberService {

    @Resource(name="memberDAO")
    private MemberDAO memberDAO;

    @Override
    public List&lt;MemberVO&gt; selectMemberList() throws Exception {
        return memberDAO.selectMemberList();
    }
}</code></pre>
<p>- 실제 로직 처리</p>
<p>- DAO 호출</p>
<p>&nbsp;</p>
<p><b>3. DAO(DB접근)</b></p>
<p>전자정부 프레임워크는 보통 mysql 기반</p>
<pre class="bash" id="code_1773471065026"><code>@Repository("memberDAO")
public class MemberDAO extends EgovAbstractDAO {

    public List&lt;MemberVO&gt; selectMemberList() throws Exception {
        return (List&lt;MemberVO&gt;) list("memberDAO.selectMemberList");
    }
}</code></pre>
<p>- DB 쿼리 실행</p>
<p>&nbsp;</p>
<p><b>4. SQL Mapper(XML)</b></p>
<pre class="bash" id="code_1773471096547"><code>&lt;select id="selectMemberList" resultType="MemberVO"&gt;
    SELECT
        MEMBER_ID,
        MEMBER_NAME,
        EMAIL
    FROM MEMBER
&lt;/select&gt;</code></pre>
<p>- DB 쿼리 관리</p>
