<p style="text-align: center;">Nexacro</p>
<hr contenteditable="false" />
<h3 style="text-align: left;">Nexacro란?</h3>
<p>웹 기반 업무 시스템의 UI를 빠르게 만들기 위한 기업용 개발 플랫폼</p>
<p>-&gt; 기업용 프로그램 화면 만드는 도구</p>
<hr contenteditable="false" />
<h3>사용하는 이유</h3>
<p>기업 시스템에서는</p>
<p>- 데이터 입력 화면 많음</p>
<p>- 조회 화면 많음</p>
<p>- 테이블 형태 많음</p>
<p>&nbsp;</p>
<p>이걸 웹 + 앱처럼 빠르게 만들기 위해 Nexacro 사용한다.</p>
<p>&nbsp;</p>
<p><b>장점</b></p>
<p>- UI 개발 속도 빠름</p>
<p>- Grid(테이블) 기능</p>
<p>- 데이터 바인딩 간편</p>
<p>- 대규모 업무 시스템에 적합</p>
<hr contenteditable="false" />
<h3>특징</h3>
<p>- 기업용 UI 플랫폼 : 내부 업무 시스템에 특화</p>
<p>- Grid 강력 : 데이터 테이블 처리 좋음</p>
<p>- 데이터 바인딩 : DB 데이터 화면 연결 쉬움</p>
<hr contenteditable="false" />
<h3>구조</h3>
<p>Nexacro는 보통 전자정부 프레임워크 + Spring + Nexacro 구조로 사용</p>
<p>&nbsp;</p>
<p>[화면] Nexacro -(API 통신)&gt;</p>
<p>[서버] Spring / 전자정부 프레임워크 -&gt;</p>
<p>[DB] Oracle / MySQL</p>
<p>&nbsp;</p>
<p><b>Nexacro(화면) &rarr; Spring Controller &rarr; Service &rarr; DAO &rarr; DB</b></p>
<hr contenteditable="false" />
<h3 style="text-align: center;">한줄 정리</h3>
<p style="text-align: center;">Nexacro는 기업용 업무 시스템의 UI를 빠르게 개발하기 위한 RIA기반 플랫폼입니다. 주로 전자정부 프레임워크나 Spring 서버와 연동하여 공공기관이나 기업 내부 ERP 시스템 개발에 사용됩니다.</p>
