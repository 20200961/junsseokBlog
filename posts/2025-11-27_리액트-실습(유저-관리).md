<p style="text-align: center;">리액트를 사용한 유저관리</p>
<hr contenteditable="false" />
<p style="text-align: center;">구현 기능은 유저 목록, 유저등록, 유저 상세정보, 에러 페이지 이렇게 구성하였다.</p>

<h3>1. 유저 등록(UserRegister.jsx)</h3>
<p><figure class="imageblock alignCenter"><span><img height="566" src="https://blog.kakaocdn.net/dn/yhAM1/dJMcadNWqpx/Vq0daOU8ttG8nNyMjpGjm1/img.png" width="361" /></span></figure>
</p>
<p>(1) 상태관리로 입력값 저장</p>
<pre class="bash" id="code_1764232491755"><code>const [name, setName] = useState("");
const [age, setAge] = useState("");
const [online, setOnline] = useState("Offline");</code></pre>
<p>-&gt; 입력&nbsp;폼의&nbsp;이름,&nbsp;나이,&nbsp;온라인&nbsp;상태를&nbsp;state로&nbsp;관리</p>
<p>&nbsp;</p>
<p>(2) 입력값 변경처리(onchange)</p>
<pre class="bash" id="code_1764232523363"><code>const handleChangeName = (ev) =&gt; {
  setName(ev.target.value);
};

const handleChangeAge = (ev) =&gt; {
  setAge(ev.target.value);
};

const handleChangeOnline = (ev) =&gt; {
  setOnline(ev.target.value);
};</code></pre>
<p>-&gt; input&nbsp;/&nbsp;select의&nbsp;값을&nbsp;가져와&nbsp;state&nbsp;갱신</p>
<p>&nbsp;</p>
<p>(3) 신규 유저 객체 생성</p>
<pre class="bash" id="code_1764232583363"><code>const newUser = {
  id: users.length &gt; 0 ? users[users.length - 1].id + 1 : 1,
  name: name,
  age: parseInt(age),
  online: online
};</code></pre>
<p>-&gt; 기존&nbsp;users&nbsp;배열을&nbsp;기반으로&nbsp;고유&nbsp;id&nbsp;생성</p>
<p>&nbsp;</p>
<p>(4) 상위 컴포넌트로 데이터 전달</p>
<pre class="bash" id="code_1764232640778"><code>addUser(newUser);</code></pre>
<p>-&gt; props로&nbsp;전달받은&nbsp;addUser&nbsp;함수&nbsp;호출</p>
<p>-&gt; 상태&nbsp;끌어올리기(Lifting&nbsp;State&nbsp;Up)&nbsp;구조</p>

<h3>4. app.jsx 메인페이지</h3>
<p>(1) 유저 추가 함수 정의</p>
<pre class="bash" id="code_1764233256178"><code>const addUser = (newUser) =&gt; {
  setUsers([...users, newUser]);
};</code></pre>
<p>-&gt; 새로운&nbsp;유저&nbsp;객체를&nbsp;기존&nbsp;users&nbsp;배열에&nbsp;추가</p>
<p>&nbsp;</p>
<p>(2) BrowserRouter로&nbsp;라우팅&nbsp;설정</p>
<pre class="bash" id="code_1764233288274"><code>&lt;BrowserRouter&gt;</code></pre>
<p>-&gt; React&nbsp;Router의&nbsp;최상위&nbsp;컴포넌트</p>
<p><br />(3) 라우터 설정</p>
<pre class="bash" id="code_1764233325714"><code>&lt;Routes&gt;
  &lt;Route path='/' element={&lt;UserList users={users} /&gt;} /&gt;
  &lt;Route path='/user/:id' element={&lt;UserDetail users={users} /&gt;} /&gt;
  &lt;Route path='/user' element={&lt;UserRegister users={users} addUser={addUser} /&gt;} /&gt;
  &lt;Route path='*' element={&lt;NotFound /&gt;} /&gt;
&lt;/Routes&gt;</code></pre>
<p>&nbsp;</p>
<p>각 라우트 역할<br />-&nbsp;/ <br />&rarr;&nbsp;유저&nbsp;목록&nbsp;페이지 <br />&rarr;&nbsp;UserList에&nbsp;users&nbsp;전달 <br /><br />-&nbsp;/user <br />&rarr;&nbsp;유저&nbsp;등록&nbsp;페이지 <br />&rarr;&nbsp;users,&nbsp;addUser&nbsp;전달 <br /><br />-&nbsp;/user/:id <br />&rarr;&nbsp;유저&nbsp;상세&nbsp;페이지 <br />&rarr;&nbsp;URL&nbsp;파라미터&nbsp;기반&nbsp;동적&nbsp;라우팅 <br /><br />-&nbsp;* <br />&rarr;&nbsp;존재하지&nbsp;않는&nbsp;경로&nbsp;처리&nbsp;(404&nbsp;페이지)</p>
