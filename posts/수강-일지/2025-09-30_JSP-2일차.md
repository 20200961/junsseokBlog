<p style="text-align: left;">1. EL 구문</p>
<p style="text-align: left;">(1) 기존&nbsp;방식대로&nbsp;JSP문법을&nbsp;사용하여&nbsp;각&nbsp;scope의&nbsp;데이터를&nbsp;출력</p>
<p style="text-align: left;">- request.getAttribute(), session.getAttribute() 등을 이용해 데이터를 가져옴<br />- 가져온 객체는 다시 getter 메서드를 호출해야 실제 값을 출력할 수 있음<br />- 코드가 길고 가독성이 떨어짐</p>
<pre class="bash" id="code_1759219545450"><code>&lt;%
        //request scope
        String classRoom = (String)request.getAttribute("classRoom");
        Person student = (Person)request.getAttribute("student");
        
        //session scope
        String academy = (String)session.getAttribute("academy");
        Person lecture = (Person)session.getAttribute("lecture");
%&gt;
&lt;p&gt;
        학원명 : &lt;%=academy%&gt;
        강의장 : &lt;%=classRoom%&gt;
        강사 : &lt;%=lecture.getName()%&gt;, &lt;%=lecture.getAge()%&gt;, &lt;%=lecture.getGender()%&gt; &lt;br&gt;&lt;br&gt;

        수강생 정보
        &lt;ul&gt;
            &lt;li&gt;이름 : &lt;%=student.getName()%&gt;&lt;/li&gt;
            &lt;li&gt;나이 : &lt;%=student.getAge()%&gt;&lt;/li&gt;
            &lt;li&gt;성별 : &lt;%=student.getGender()%&gt;&lt;/li&gt;
        &lt;/ul&gt;
&lt;/p&gt;</code></pre>
<p style="text-align: left;">&nbsp;</p>
<p style="text-align: left;">&nbsp;</p>
<p style="text-align: left;">(2) EL문법을&nbsp;이용해서&nbsp;보다&nbsp;쉽게&nbsp;scope의&nbsp;값을&nbsp;출력할&nbsp;수&nbsp;있음</p>
<p style="text-align: left;">-&nbsp;${키값}&nbsp;형태로&nbsp;간단하게&nbsp;값&nbsp;출력&nbsp;가능 <br />-&nbsp;getter를&nbsp;직접&nbsp;호출할&nbsp;필요&nbsp;없이,&nbsp;EL이&nbsp;내부적으로&nbsp;자동&nbsp;접근 <br />-&nbsp;scope를&nbsp;구분하지&nbsp;않고(page&nbsp;&rarr;&nbsp;request&nbsp;&rarr;&nbsp;session&nbsp;&rarr;&nbsp;application&nbsp;순서)&nbsp;검색하여&nbsp;값&nbsp;반환</p>
<pre class="bash" id="code_1759219592313"><code>&lt;p&gt;
        학원명 : ${academy}
        강의장 : ${classRoom}
        강사 : ${lecture.name}, ${lecture.age}, ${lecture.gender} &lt;br&gt;&lt;br&gt;

        수강생 정보
        &lt;ul&gt;
            &lt;li&gt;이름 : ${student.name}&lt;/li&gt;
            &lt;li&gt;나이 : ${student.age}&lt;/li&gt;
            &lt;li&gt;성별 : ${student.gender}&lt;/li&gt;
        &lt;/ul&gt;
&lt;/p&gt;</code></pre>
<p style="text-align: left;">&nbsp;</p>
<p style="text-align: left;">&nbsp;</p>
<p style="text-align: left;">(3) scope의 키가 동일할 경우</p>
<p style="text-align: left;">- EL은&nbsp;가장&nbsp;좁은&nbsp;범위(page&nbsp;&rarr;&nbsp;request&nbsp;&rarr;&nbsp;session&nbsp;&rarr;&nbsp;application)부터&nbsp;값을&nbsp;찾음 <br />- 특정 scope를 지정해서 접근 가능: ${pageScope.scope}, ${requestScope.scope}, ${sessionScope.scope}, ${applicationScope.scope<br /><br /></p>
<p style="text-align: left;">&nbsp;</p>
<p style="text-align: left;">(4) 직접&nbsp;scope를&nbsp;지정해서&nbsp;접근도&nbsp;가능</p>
<pre class="bash" id="code_1759219719521"><code>&lt;% 	//page copre에 값을 저장할 때
    	pageContext.setAttribute("scope", "page"); 
    %&gt;
    
    pageScope : ${scope}, ${pageScope.scope} &lt;br&gt;
    requestScope : ${requestScope.scope} &lt;br&gt;
    sessionScope : ${sessionScope.scope} &lt;br&gt;
    applicationScope : ${applicationScope.scope} &lt;br&gt;</code></pre>

<p>2. 연산</p>
<p>(1) 산술 연산</p>
<p>-&nbsp;EL은&nbsp;기본적인&nbsp;사칙연산과&nbsp;나머지&nbsp;연산을&nbsp;지원함. <br />- ${big + small}, ${big - small}, ${big * small}, ${big / small}/는&nbsp;div,&nbsp;%는&nbsp;mod&nbsp;키워드로도&nbsp;표현&nbsp;가능.</p>
<pre class="bash" id="code_1759219922985"><code>10 + 3 = ${big + small} &lt;br&gt;
10 - 3 = ${big - small} &lt;br&gt;
10 * 3 = ${big * small} &lt;br&gt;
10 / 3 = ${big / small} 또는 ${big div small}&lt;br&gt;
10 % 3 = ${big % small} 또는 ${big mod small}&lt;br&gt;</code></pre>
<p>&nbsp;</p>
<p>(2) 대소 비교</p>
<p>-&nbsp;숫자&nbsp;비교도&nbsp;EL에서&nbsp;바로&nbsp;가능. <br />-&nbsp;&gt;,&nbsp;&lt;,&nbsp;&gt;=,&nbsp;&lt;=와&nbsp;같은&nbsp;기호&nbsp;연산자뿐만&nbsp;아니라&nbsp;gt,&nbsp;lt,&nbsp;ge,&nbsp;le라는&nbsp;키워드로도&nbsp;표현&nbsp;가능</p>
<pre class="bash" id="code_1759219936025"><code>10 + 3 = ${big + small} &lt;br&gt;
10 - 3 = ${big - small} &lt;br&gt;
10 * 3 = ${big * small} &lt;br&gt;
10 / 3 = ${big / small} 또는 ${big div small}&lt;br&gt;
10 % 3 = ${big % small} 또는 ${big mod small}&lt;br&gt;</code></pre>
<p>&nbsp;</p>
<p>(3) 동등 비교</p>
<p>-&nbsp;EL에서&nbsp;==와&nbsp;!=는&nbsp;자바의&nbsp;equals()와&nbsp;유사하게&nbsp;동작. <br />-&nbsp;문자열&nbsp;비교도&nbsp;지원하며,&nbsp;eq,&nbsp;ne&nbsp;키워드를&nbsp;사용할&nbsp;수도&nbsp;있음.</p>
<pre class="bash" id="code_1759219950040"><code>10 &amp;gt; 3 = ${big &gt; small} 또는 ${big gt small} &lt;br&gt;
10 &amp;lt; 3 = ${big &lt; small} 또는 ${big lt small} &lt;br&gt;
10 &amp;gt;= 3 = ${big &gt;= small} 또는 ${big ge small} &lt;br&gt;
10 &amp;lt;= 3 = ${big &lt;= small} 또는 ${big le small} &lt;br&gt;</code></pre>
<p>&nbsp;</p>

<p>3. include</p>
<p>&nbsp;</p>
<p>(1) include 지시어 &lt;%@ include ... %&gt; -&gt; 정적 include <br />-&nbsp;컴파일&nbsp;시점에&nbsp;포함되는&nbsp;방식.&nbsp;(.jsp가&nbsp;.java로&nbsp;변환될&nbsp;때&nbsp;함께&nbsp;합쳐진다) <br />-&nbsp;포함된&nbsp;페이지의&nbsp;변수나&nbsp;선언을&nbsp;현재&nbsp;페이지에서도&nbsp;그대로&nbsp;사용할&nbsp;수&nbsp;있음</p>
<pre class="bash" id="code_1759220223192"><code>&lt;%@ include file="footer.jsp" %&gt;</code></pre>
<p>&nbsp;</p>
<p>(2) jsp:include 표준 액션 태그 &lt;jsp:include ... /&gt; -&gt; 동적 include<br />- 실행(런타임) 시점에 포함되는 방식<br />- 포함된 페이지의 변수를 현재 페이지에서 공유하지 않음 -&gt; 독립적 실행</p>
<pre class="bash" id="code_1759220231248"><code>&lt;jsp:include page="footer.jsp" /&gt;</code></pre>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p>(3) jsp:forward<br />- 서버 내부에서 실행되는 "내부 리다이렉트" 기능<br />- 현재 JSP에서 처리 중인 요청(request)과 응답(response)을 다른 JSP/Servlet으로 그대로 전달하고, 현재 페이지의 처리를 즉시 종료<br />- 브라우저는 요청이 다른 페이지로 넘어갔다는 사실을 모름 -&gt; 따라서 URL은 변경되지 않음</p>
<p>&nbsp;</p>
<pre class="bash" id="code_1759220354776"><code>&lt;jsp:forward page="footer.jsp" /&gt;</code></pre>
<p>&nbsp;</p>
<p>-&nbsp;클라이언트는&nbsp;전혀&nbsp;알&nbsp;수&nbsp;없고&nbsp;서버&nbsp;내부에서만&nbsp;처리&nbsp;흐름이&nbsp;변경됨 <br />-&nbsp;요청(request)과&nbsp;응답(response)이&nbsp;그대로&nbsp;전달되므로&nbsp;&nbsp;request&nbsp;영역에&nbsp;담긴&nbsp;데이터도&nbsp;함께&nbsp;넘어감 <br />-&nbsp;sendRedirect와&nbsp;달리&nbsp;네트워크&nbsp;재요청이&nbsp;발생하지&nbsp;않음</p>

<p>3. JSTL</p>
<p>(1)&nbsp;변수&nbsp;선언&nbsp;및&nbsp;삭제(c:set,&nbsp;c:remove) <br />-&nbsp;c:set&nbsp;:&nbsp;변수를&nbsp;선언하고&nbsp;초기화.&nbsp;scope&nbsp;속성을&nbsp;지정하면&nbsp;해당&nbsp;내장&nbsp;객체에&nbsp;저장됨 <br />-&nbsp;c:remove&nbsp;:&nbsp;변수를&nbsp;제거.&nbsp;scope를&nbsp;지정하지&nbsp;않으면&nbsp;모든&nbsp;scope에서&nbsp;삭제 <br />-&nbsp;EL로&nbsp;바로&nbsp;접근&nbsp;가능</p>
<pre class="bash" id="code_1759220652887"><code>&lt;c:set var="num1" value="10" /&gt;
&lt;c:set var="num2" value="20" scope="request" /&gt;

num1 : ${num1} &lt;br&gt;
num2 : ${num2} &lt;br&gt;

&lt;c:remove var="num1" /&gt;
삭제 후 num1 : ${num1}   &lt;!-- null 출력 --&gt;</code></pre>
<p><br /><br />(2)&nbsp;출력&nbsp;(c:out) <br />-&nbsp;${}와&nbsp;비슷하지만,&nbsp;값이&nbsp;null일&nbsp;때&nbsp;기본값&nbsp;출력&nbsp;가능 <br />-&nbsp;escapeXml&nbsp;옵션으로&nbsp;HTML&nbsp;태그를&nbsp;이스케이프&nbsp;처리&nbsp;여부&nbsp;선택&nbsp;가능</p>
<pre class="bash" id="code_1759220679135"><code>&lt;c:set var="msg" value="&lt;b&gt;Hello JSP&lt;/b&gt;" /&gt;

기본 EL 출력 : ${msg} &lt;br&gt;
c:out 출력(escapeXml=true) : &lt;c:out value="${msg}" /&gt; &lt;br&gt;
c:out 출력(escapeXml=false) : &lt;c:out value="${msg}" escapeXml="false"/&gt; &lt;br&gt;</code></pre>
<p><br /><br />(3)&nbsp;조건문&nbsp;(c:if,&nbsp;c:choose) <br />-&nbsp;c:if&nbsp;:&nbsp;단순&nbsp;조건&nbsp;분기&nbsp;(자바의&nbsp;if와&nbsp;동일) <br />-&nbsp;c:choose&nbsp;:&nbsp;다중&nbsp;분기&nbsp;(자바의&nbsp;switch-case와&nbsp;유사) <br />-&nbsp;조건식은&nbsp;EL&nbsp;구문으로&nbsp;작성</p>
<pre class="bash" id="code_1759220685055"><code>&lt;c:set var="age" value="25" /&gt;

&lt;c:if test="${age &gt;= 20}"&gt;
  성인입니다.
&lt;/c:if&gt;

&lt;c:choose&gt;
  &lt;c:when test="${age &lt; 13}"&gt;어린이&lt;/c:when&gt;
  &lt;c:when test="${age &lt; 20}"&gt;청소년&lt;/c:when&gt;
  &lt;c:otherwise&gt;성인&lt;/c:otherwise&gt;
&lt;/c:choose&gt;</code></pre>
<p><br /><br />(4)&nbsp;반복문&nbsp;(c:forEach) <br />-&nbsp;카운터&nbsp;반복:&nbsp;begin,&nbsp;end,&nbsp;step을&nbsp;지정하여&nbsp;반복 <br />-&nbsp;배열/컬렉션&nbsp;반복:&nbsp;items&nbsp;속성으로&nbsp;리스트나&nbsp;배열을&nbsp;받아&nbsp;순차&nbsp;접근 <br />-&nbsp;varStatus로&nbsp;현재&nbsp;반복&nbsp;상태(index,&nbsp;count&nbsp;등)&nbsp;확인&nbsp;가능</p>
<pre class="bash" id="code_1759220692808"><code>&lt;c:forEach var="i" begin="1" end="5"&gt;
  반복 : ${i} &lt;br&gt;
&lt;/c:forEach&gt;

&lt;c:set var="colors" value="red,blue,green" /&gt;
&lt;ul&gt;
  &lt;c:forTokens var="c" items="${colors}" delims=","&gt;
    &lt;li style="color:${c}"&gt;${c}&lt;/li&gt;
  &lt;/c:forTokens&gt;
&lt;/ul&gt;</code></pre>
<p><br /><br />(5)&nbsp;문자열&nbsp;분리&nbsp;반복&nbsp;(c:forTokens) <br />-&nbsp;문자열을&nbsp;구분자(delims)&nbsp;기준으로&nbsp;나눠&nbsp;반복&nbsp;처리 <br />-&nbsp;자바의&nbsp;String.split()과&nbsp;유사한&nbsp;역할</p>
<pre class="bash" id="code_1759220698479"><code>&lt;c:forTokens var="d" items="${device}" delims=",/."&gt;
	&lt;li&gt;${d}&lt;/li&gt;
&lt;/c:forTokens&gt;</code></pre>
<p><br /><br />(6)&nbsp;URL&nbsp;처리&nbsp;(c:url,&nbsp;c:param) <br />-&nbsp;c:url&nbsp;:&nbsp;요청&nbsp;URL을&nbsp;생성할&nbsp;때&nbsp;사용 <br />-&nbsp;내부에서&nbsp;c:param으로&nbsp;쿼리스트링을&nbsp;정의할&nbsp;수&nbsp;있음 <br />-&nbsp;URL&nbsp;조립이&nbsp;편리하고,&nbsp;EL과&nbsp;함께&nbsp;활용&nbsp;가능</p>
<pre class="bash" id="code_1759220725799"><code>&lt;c:url var="link" value="list.do"&gt;
  &lt;c:param name="page" value="1"/&gt;
  &lt;c:param name="size" value="10"/&gt;
&lt;/c:url&gt;

&lt;a href="${link}"&gt;목록 보기&lt;/a&gt;</code></pre>

<p>4. JSTL fmt 태그</p>
<p>(1) 숫자 포맷 (fmt:formatNumber)<br />-&nbsp;숫자&nbsp;데이터를&nbsp;원하는&nbsp;표시&nbsp;형식으로&nbsp;변환할&nbsp;수&nbsp;있음 <br />-&nbsp;groupingUsed="true|false"&nbsp;:&nbsp;천&nbsp;단위&nbsp;구분자(,)&nbsp;사용&nbsp;여부 <br />-&nbsp;type="number|percent|currency"&nbsp;:&nbsp;출력&nbsp;형식 <br />-&nbsp;currencySymbol="문자"&nbsp;:&nbsp;통화&nbsp;심볼&nbsp;지정 <br />-&nbsp;currencyCode="KRW,&nbsp;USD,&nbsp;JPY&nbsp;등"&nbsp;:&nbsp;국제&nbsp;통화&nbsp;코드&nbsp;기반&nbsp;출력</p>
<pre class="bash" id="code_1759220914583"><code>&lt;c:set var="num1" value="123456789" /&gt;
&lt;c:set var="num2" value="0.75" /&gt; 
&lt;c:set var="num3" value="50000" /&gt;

출력 : ${num1}&lt;br&gt;
세 자리 구분 출력 : &lt;fmt:formatNumber value="${num1}" /&gt; &lt;br&gt;
세 자리 구분 없이 : &lt;fmt:formatNumber value="${num1}" groupingUsed="false" /&gt; &lt;br&gt;

퍼센트(기본) : &lt;fmt:formatNumber value="${num2}" type="percent" /&gt; &lt;br&gt;
퍼센트(소수 1자리) : 
&lt;fmt:formatNumber value="${num2}" type="percent" minFractionDigits="1" maxFractionDigits="1" /&gt; &lt;br&gt;

통화(심볼) : &lt;fmt:formatNumber value="${num3}" type="currency" currencySymbol="$" /&gt; &lt;br&gt;
통화(코드) : &lt;fmt:formatNumber value="${num3}" type="currency" currencyCode="KRW" /&gt; &lt;br&gt;</code></pre>
<p>&nbsp;</p>
<p>(2) 날짜&nbsp;포맷&nbsp;(fmt:formatDate)</p>
<p>-&nbsp;java.util.Date&nbsp;객체를&nbsp;다양한&nbsp;포맷으로&nbsp;출력 <br />-&nbsp;type="date|time|both"&nbsp;:&nbsp;날짜/시간&nbsp;출력&nbsp;구분 <br />-&nbsp;dateStyle|timeStyle="short|medium|long|full"&nbsp;:&nbsp;스타일&nbsp;지정 <br />-&nbsp;pattern="yyyy-MM-dd&nbsp;HH:mm:ss"&nbsp;:&nbsp;직접&nbsp;포맷&nbsp;지정</p>
<pre class="bash" id="code_1759220965176"><code>&lt;c:set var="current" value="&lt;%=new java.util.Date() %&gt;" /&gt;

&lt;ul&gt;
  &lt;li&gt;현재 날짜 : &lt;fmt:formatDate value="${current}" type="date"/&gt; &lt;/li&gt;
  &lt;li&gt;현재 시간 : &lt;fmt:formatDate value="${current}" type="time"/&gt; &lt;/li&gt;
  &lt;li&gt;날짜+시간 : &lt;fmt:formatDate value="${current}" type="both"/&gt; &lt;/li&gt;
  &lt;li&gt;medium : &lt;fmt:formatDate value="${current}" type="both" dateStyle="medium" timeStyle="medium"/&gt; &lt;/li&gt;
  &lt;li&gt;long : &lt;fmt:formatDate value="${current}" type="both" dateStyle="long" timeStyle="long"/&gt; &lt;/li&gt;
  &lt;li&gt;short : &lt;fmt:formatDate value="${current}" type="both" dateStyle="short" timeStyle="short"/&gt; &lt;/li&gt;
  &lt;li&gt;full : &lt;fmt:formatDate value="${current}" type="both" dateStyle="full" timeStyle="full"/&gt; &lt;/li&gt;
  &lt;li&gt;pattern : 
    &lt;fmt:formatDate value="${current}" type="both" 
      pattern="yyyy년 MM월 dd일 E요일 a HH:mm:ss"/&gt; 
  &lt;/li&gt;
&lt;/ul&gt;</code></pre>

<p>5. JSTL&nbsp;Function&nbsp;라이브러리</p>
<p>-&nbsp;JSTL&nbsp;fn&nbsp;라이브러리는&nbsp;EL(Expression&nbsp;Language)&nbsp;내에서&nbsp;문자열,&nbsp;배열,&nbsp;컬렉션을&nbsp;조작할&nbsp;수&nbsp;있는&nbsp;함수를&nbsp;제공함 <br />-&nbsp;대부분의&nbsp;기능은&nbsp;자바&nbsp;기본&nbsp;메서드로&nbsp;대체&nbsp;가능하지만,&nbsp;EL에서&nbsp;바로&nbsp;사용할&nbsp;수&nbsp;있다는&nbsp;장점이&nbsp;있음</p>
<pre class="bash" id="code_1759221147559"><code>&lt;%@ taglib prefix="fn" uri="http://java.sun.com/jsp/jstl/functions" %&gt;
&lt;c:set var="str" value="How are you?" /&gt;

문자열 길이 : ${str.length()} 또는 ${fn:length(str)} &lt;br&gt;
"are" 시작 인덱스 : ${fn:indexOf(str, "are")} &lt;br&gt;
대문자 변환 : ${fn:toUpperCase(str)} &lt;br&gt;
소문자 변환 : ${fn:toLowerCase(str)} &lt;br&gt;
